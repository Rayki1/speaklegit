GOOGLE LOGIN + MONGODB ATLAS QUICK START

1) Frontend terminal (project root):
   npm install
   npm run dev

2) Backend terminal:
   cd backend
   npm install
   node server.js

Required backend/.env values:
- MONGODB_URI=your_mongodb_atlas_connection_string
- MONGODB_DB_NAME=speaks_app
- JWT_SECRET=your_secret_here
- GOOGLE_CLIENT_ID=your_google_client_id_here
- FRONTEND_URL=http://localhost:5173

Google Cloud OAuth client must include:
- Authorized JavaScript origin: http://localhost:5173

Notes:
- This project now uses MongoDB Atlas only.
- No MySQL or SQL import file is needed.
